# Video Compilation Page

## Descrição

Este projeto é uma página de compilação de vídeos desenvolvida com HTML. A página apresenta três dos momentos mais marcantes da história recente do São Paulo Futebol Clube, cada um representando uma década diferente do século XXI.

Os vídeos são incorporados diretamente do YouTube e acompanhados por descrições que contextualizam cada conquista.

## Objetivo

O objetivo do projeto é praticar conceitos fundamentais de HTML5, incluindo:

- Estruturação semântica de páginas
- Organização de conteúdo em seções
- Uso de títulos e parágrafos
- Incorporação de vídeos com `iframe`

## Estrutura do Projeto

A página contém as seguintes seções:

- **Cabeçalho (head)**
  - Definição de charset
  - Título da página

- **Corpo (body)**
  - Título principal
  - Texto introdutório
  - Seção sobre o Mundial de Clubes de 2005
  - Seção sobre a Copa Sul-Americana de 2012
  - Seção sobre a Copa do Brasil de 2023
  - Vídeos incorporados do YouTube

## Tecnologias Utilizadas

- HTML5

## Como Executar o Projeto

1. Clone este repositório:

   ```bash
   git clone <https://github.com/dev-alexandregarcia/site-de-agencia-de-viagens-freeCodeCamp.git>
   ```

## Conceitos Praticados

- HTML5 semântico
- Elementos `main` e `section`
- Hierarquia de títulos (`h1` e `h2`)
- Parágrafos (`p`)
- Incorporação de conteúdo externo com `iframe`
- Organização de conteúdo multimídia

## Conteúdo da Página

### O Tri Mundial (2005)

Vitória histórica sobre o Liverpool por 1 a 0, garantindo ao São Paulo seu terceiro título mundial.

### Copa Sul-Americana (2012)

Conquista internacional invicta diante do Tigre, da Argentina.

### Copa do Brasil (2023)

Título inédito conquistado contra o Flamengo, completando a galeria de troféus do clube.
